## Overview

Personal portfolio built as a bilingual (English / Portuguese) single-page application with a full snap-scroll layout and entrance animations throughout. It presents who I am, the projects I've built, and how to get in touch.

## Sections

| Section | Description |
|---|---|
| **Intro** | Typewriter cycling through my roles (Data Analyst, Data Engineer, AI Enthusiast) + a short bio |
| **Projects** | Responsive card grid showcasing selected work with images, descriptions, and tech tags |
| **Contact** | GitHub and LinkedIn links, with a resume placeholder (coming soon) |

A floating pill nav at the bottom allows jumping between sections, toggling the language, and a dot indicator on the left tracks the current section on desktop.
